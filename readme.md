Black and White Spa.

En el siguiente trabajo se requiere desarrollar un sitio en cual se pueda ingresar el link de una imagen de internet para procesarla y cambiar sus propiedades como el color y su tamaño. Para realizar esta tarea se necesita levantar un servidor, estableces carpetas públicas y construir archivos independientes de estilos, lógica y estructura. 

En el siguiente trabajo se utilizaron las siguiuentes tecnologías:

-HTML y CSS.
-NodeJS.
-Módulo Jimp.
-Módulo UUID.
-Módulo Express.
